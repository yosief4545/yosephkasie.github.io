Enhancement Two: Algorithms and Data Structures

This artifact was based on my CS 300 Project Two course planner. The program loads course data from a file, stores it in a binary search tree, prints the courses in alphanumeric order, and allows users to search for a course and its prerequisites.

In the enhanced version, I improved file validation, user input handling, memory management, and overall maintainability. These improvements made the artifact more reliable and more professional while preserving its original purpose.

This enhancement demonstrates my skills in problem-solving, binary search trees, searching, traversal, input validation, and software improvement.
