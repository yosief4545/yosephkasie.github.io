Enhancement Two: Algorithms and Data Structures

This artifact was based on my CS 300 Project Two course planner. The original program loaded course data from a file, stored it in a binary search tree, printed the courses in alphanumeric order, and allowed users to search for a course and its prerequisites. I selected this artifact because it clearly demonstrates my ability to apply algorithms and data structures to solve a practical problem in an organized and efficient way.

In the enhanced version, I improved file validation, user input handling, memory management, and overall maintainability. These changes made the artifact more reliable and more professional while preserving its original purpose. This process showed the importance of selecting appropriate data structures, improving program quality, and thinking carefully about how design decisions affect performance and usability.

This enhancement demonstrates my skills in problem-solving, binary search trees, searching, traversal, input validation, maintainability, and software improvement. It also reflects my growth in applying algorithmic principles to real programming tasks.
